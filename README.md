## The U-Net model is a Convolutional Neural Network (CNN) characterized by its symmetric "U" shape:

## Contracting Path (Encoder): Captures context through downsampling and feature extraction.

## Expansive Path (Decoder): Enables precise localization through upsampling.

## Skip Connections: Links the encoder and decoder to preserve spatial information and fine-grained details that are often lost during pooling.
